An **interactive Streamlit dashboard** to analyze sales, profit, and customer performance using the Global Superstore dataset.  
This project demonstrates business intelligence (BI) visualization, KPI analysis, and interactive filtering.

---

## Features

- Calculate **Total Sales** and **Total Profit**  
- Display **Top 5 Customers by Sales**  
- Filter data by **Region, Category, and Sub-Category** (can be added to Streamlit app)  
- Interactive and easy-to-use dashboard

---

## Dataset

- Source: [Global Superstore Dataset](https://www.kaggle.com/datasets/juhi1994/global-superstore-dataset)  
- CSV file: `Global_Superstore.csv`  
- Make sure the CSV is in the same folder as `app.py` to run the app locally
